# refboard_v1

Print [refboard_v1_letter.pdf](./refboard_v1_letter.pdf) at 100% on US Letter paper.

Quick checks:
- Verify the black scale bar measures exactly 50 mm.
- Do not enable `Fit to page`, `Scale to fit`, or any auto-resize option.
- Keep the board flat and place it in the same plane as the pattern piece.
